### 插入模式
按 i  insert
### 普通模式
按 Esc

表达方式 ps: ctrl + v ==  ^V  ||  Ctrl-v  ||  <C-V>

### 替换模式
按 r  replace

### 其他
按 v  visual 可视化模式 
按 shift-v -line  行可视化模式
按 ctrl-v  -block  块可视化模式  
按 :  命令模式


: w 表示写入文件即保存
: q 表示退出 vim / 关闭当前窗口
: qa 表示退出所有窗口
：wq表示写入文件并退出 vim
: q!强制退出并且不保存
：sp 表示水平分屏,打开同个文件新窗口 :vsp 表示垂直分屏,打开同个文件新窗口


: tabnew 表示新建一个 tab 页
: tabclose 表示关闭当前 tab 页
: tabn 表示切换到下一个 tab 页
: tabp 表示切换到上一个 tab 页

hjkl 表示方向键
w 表示向右移动一个字符
b 表示向左移动一个字符
0 表示移动到行首
$ 表示移动到行尾
^ 表示向行首移动第一个非空格字符
ctrl + u 表示向上移动一行
ctrl + d 表示向下移动一行

G 表示移动到文件末尾
gg 表示移动到文件开头

L 屏幕显示最后一行
H 屏幕显示第一行
M 屏幕显示当前行

d 删除
dd 删除一行

c 删除并进入插入模式
cc + j/k 删除并进入插入模式

v 可视化模式  +   y 表示复制选中区域  + p 表示粘贴

u 撤销上一次操作


以上均可自由组合
