## 常用命令

### 查看帮助
man
-h


### 切换 
cd
可绝对路径：
cd /home/ghs/Desktop/mark
可相对路径：
cd ../
..指的是上一级目录
.指的是当前目录

### 查看当前目录
pwd


### 查看目录下的文件
ls

### 查看当前目录
pwd

### 清屏
clear



### 创建目录
mkdir


### 删除目录
rmdir


### 创建文件
touch


### 删除文件
rm
# 单个源：删除文件/目录
rm [选项] 源文件/目录
# 多个源：批量删除
rm [选项] 源1 源2 源3 ...

### 复制文件
cp
# 单个源：复制文件/目录到目标
cp [选项] 源文件/目录 目标文件/目录
# 多个源：仅能复制到「目标目录」（目标必须是已存在的目录）
cp [选项] 源1 源2 源3 ... 目标目录


### 移动文件
mv
# 单个源：移动/重命名
mv [选项] 源文件/目录 目标文件/目录
# 多个源：仅能移动到「目标目录」（目标必须是已存在的目录）
mv [选项] 源1 源2 源3 ... 目标目录


### 打开文件
open



### 查看文件内容
cat

### 执行当前目录下的文件
./server


### 重定向

< 输入重定向
> 输出重定向

cat 默认从终端读取内容并输出到终端


cat < file.txt //意为读取file.txt文件的内容并输出到终端


>>追加输出重定向
cat < file.txt >> file.txt //意为将file.txt文件的内容追加到file.txt文件的末尾

### 管道符
   左程序  |  右程序     把左侧内容作为右侧程序的输入

### 切换到root用户
sudo su 

### sudo 命令
sudo 命令用于以超级用户（root）权限执行命令。
但有时无法成功，注意
sudo echo 500 > brightness
无法成功是因为sudo 只给到了echo 命令的权限，而没有给到重定向 > 权限

### 退出
exit

###  tee 命令
tee 命令用于将标准输入的数据写入到文件中，同时也会将数据显示到终端。



### 打开文件
xdg-open

与 ./server 不同，xdg-open 可以打开任意文件类型，包括图片、文档、视频等。而且是以默认应用打开文件。
